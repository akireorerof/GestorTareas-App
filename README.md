## GestorTareas 📝

## Estudiante

## Descripción
Aplicación Android nativa para gestionar tareas personales.
Permite crear, editar, eliminar y marcar tareas como completadas.
Usa SQLite para persistencia de datos y RecyclerView para mostrar la lista.

## Tecnologías
- Java
- SQLite (CRUD)
- RecyclerView
- Material Design
- Android Studio

## Funcionalidades
- ✅ Agregar tareas
- ✅ Editar tareas
- ✅ Eliminar tareas
- ✅ Marcar como completada o pendiente
- ✅ Datos persistentes con SQLite
