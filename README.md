## Descripción
Aplicación Android nativa para gestionar tareas personales.

Permite crear, editar, eliminar, buscar y marcar tareas como completadas o pendientes.

La aplicación utiliza Firebase Firestore para el almacenamiento y sincronización de datos en tiempo real y RecyclerView para mostrar las tareas en formato de cuadrícula con diseño moderno.

## Tecnologías
Java
Firebase Firestore
RecyclerView
Material Design
Android Studio

## Funcionalidades
- ✅ Agregar tareas
- ✅ Editar tareas
- ✅ Eliminar tareas
- ✅ Marcar como completada o pendiente
- ✅ Datos almacenados en Firebase Firestore
